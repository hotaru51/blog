メモするまでもないかと思ったけど、やろうとして思い出せなかったのでメモ。
インストールしているAtomのプラグインのリストを吐き出して、そのリストを元に別のマシンのAtomに同じプラグインをインストールする。

まずはリストを吐き出したいマシン上で以下のコマンドを実行。こちらはWindowsだったのでコマンドプロンプト。

apm list --installed --bare > atom_package.txt

吐き出したリストをインストール先のマシンに移動して、以下のコマンドを実行するとまとめてインストールされる。こちらはMac上で実行

apm install --packages-file atom_package.txt 
Installing ex-mode@0.18.0 to /Users/<ユーザ名>/.atom/packages ✓
Installing monokai@0.24.0 to /Users/<ユーザ名>/.atom/packages ✓
Installing vim-mode-plus@1.30.1 to /Users/<ユーザ名>/.atom/packages ✓

ただ、GitHub経由で設定を同期する機能があるらしいので、ちゃんと使うならそっちで設定した方がいいんだろうなぁ、と…
